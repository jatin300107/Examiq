# Examiq

Examiq is a role-based quiz and assessment platform built with Flask.

## Features
- Instructor and learner roles
- Create and manage tests
- Dynamic question navigation
- Answer tracking
- Automated score calculation
- Authentication and authorization

## Tech Stack
- Python
- Flask
- SQLite
- HTML

## Purpose
This project was built to explore backend system design, role-based access control, and dynamic test handling.

## Future Improvements
- REST API support
- FastAPI version
- Analytics for test results
