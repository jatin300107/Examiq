# Examiq

Examiq is a role-based quiz and assessment platform built with Flask where instructors can create tests and learners can attempt them through a dynamic interface with automated scoring.

## Features
- Instructor and learner roles
- Create and manage tests
- Dynamic question navigation
- Answer tracking
- Automated score calculation
- Authentication and authorization

  ## Role System
  The Platform supports two roles:
  - **Learner** - can attempt tests and receive scores.
  - **Instructor** - can create and manage assessments.
  Instructor access is enabled durning registration using an instructor autorization key.

## Tech Stack
- Python
- Flask
- SQLite
- HTML
- CSS

## Purpose
This project was built to explore backend system design, role-based access control, and dynamic test handling.
